# app-pt05-async-iterator-stream

## Running


```sh
# restore dependencies
npm ci --silent
```

Make sure the [api](./../api/) is running.

Choose web or cli to run
### Web
```sh
npm run web
```
Go to [http://localhost:3001/ui](http://localhost:3001/ui)

### CLI
```sh
npm run cli
```

to run tests you can run

```sh
npm t
```

or on watch mode

```sh
npm run test:watch
```
